# ESP32-CAM-Wi-Fi-Surveillance-Car



## Overview

A Wi-Fi-enabled surveillance car built using ESP32 and ESP32-CAM modules. The vehicle can be controlled remotely through a mobile browser, while providing real-time video streaming from the onboard camera. The project combines wireless communication, embedded systems, IoT, and computer vision hardware integration.

## Features

* Real-time video streaming using ESP32-CAM
* Mobile browser-based control interface
* Wireless Wi-Fi communication
* Forward, backward, left, and right movement control
* Live monitoring from any device connected to the network
* Low-cost and portable surveillance solution
* Responsive web interface for smartphone control

## Hardware Components

* ESP32 Development Board
* ESP32-CAM Module
* L298N Motor Driver
* DC Gear Motors
* Li-ion Battery Pack
* Jumper Wires
* Wi-Fi Network

## System Architecture

1. ESP32 hosts a web server for vehicle control.
2. Mobile browser sends movement commands to the ESP32.
3. ESP32 processes commands and drives motors through the motor driver.
4. ESP32-CAM streams live video over Wi-Fi.
5. User monitors surroundings and controls the vehicle remotely.
